# Design Files

This directory is the intended home for Pencil design files for this project.

Expected files:

- `running-coach.pen`
- `pencil-lunaris.pen` only if it is actively used as a reference

Responsibilities:

- `running-coach.pen`: the only product design source of truth for sitemap, IA, wireframes, and ready-for-dev screens
- `pencil-lunaris.pen`: optional reference kit, never a competing product source of truth

Note:

When working through Pencil MCP, the editor may open and edit these paths before the `.pen`
files are visibly materialized on disk. If that happens, save or confirm the document from the
Pencil app so the files are persisted in this directory.

If a reference kit is used in active product work, keep it here so the repo contains the same
visual baseline the team is designing against.
