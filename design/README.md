# Design Files

This directory is the intended home for Pencil design files for this project.

Expected files:

- `pencil-lunaris.pen`
- `running-coach.pen`

Responsibilities:

- `pencil-lunaris.pen`: reference UI kit and component baseline used to derive the project subset
- `running-coach.pen`: product sitemap, IA, wireframes, and ready-for-dev screens

Note:

When working through Pencil MCP, the editor may open and edit these paths before the `.pen`
files are visibly materialized on disk. If that happens, save or confirm the document from the
Pencil app so the files are persisted in this directory.

If Lunaris is used in active product work, save the kit here as `design/pencil-lunaris.pen`
so the repo contains the same visual baseline the team is designing against.
