# Running Coach Web

User-facing web app for Running Coach.

This repo contains the frontend layer, not the backend. The current shipped scope is small: public product pages, privacy, and a clean foundation for the future user portal.

## Stack

- Astro
- TypeScript
- plain CSS

## Source of truth

- design: `design/running-coach.pen`
- frontend: `src/`
- tokens: `src/styles/tokens.css`

## Commands

- `npm install`
- `npm run dev`
- `npm run format`
- `npm run lint`
- `npm run build`
- `npm run preview`
